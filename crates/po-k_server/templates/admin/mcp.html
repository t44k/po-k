2. add to ~/.claude/mcp_servers.json

If the file doesn't exist yet, create it with this content. If it already has an mcpServers object, merge the po-k entry into it.

{
  "mcpServers": {
    "po-k": {
      "transport": "http",
      "url": "{{ server_url }}/mcp",
      "headers": {
        "X-Api-Key": "{% if let Some(m) = minted %}{{ m.plaintext }}{% else %}pk_...paste-the-minted-key-here{% endif %}"
      }
    }
  }
}

3. restart Claude Code

Open a fresh Claude Code session. /mcp should list po-k as connected. Try:

# Show curated topics for this team
mcp__po-k__list_topics()

# Read the maintained answer to a topic
mcp__po-k__recall_topic("auth-pattern")

# Hybrid search over events
mcp__po-k__search_sessions(query="bm25 retrieval", limit=10)
